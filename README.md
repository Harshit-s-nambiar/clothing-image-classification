# Clothing Image Classification

Deep learning based clothing image classification using CNN and VGG16 transfer learning.

---

## Project Overview

This project classifies clothing images into different categories using Deep Learning and Computer Vision techniques.

The project begins with a basic CNN model and improves performance using:
- Data Augmentation
- Early Stopping
- Transfer Learning (VGG16)

Final model accuracy achieved:
### 85%+ Accuracy
